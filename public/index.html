Bellevue LifeSpring AI Assistant — Pilot Project

Built an AI-powered chatbot for Bellevue LifeSpring, a local nonprofit serving low-income families in the greater Seattle area. The chatbot answers questions about the organization's services, programs, and how to get involved — drawing exclusively from the nonprofit's website content. Built using the Claude API with a custom system prompt, deployed with a clean embeddable chat interface that any nonprofit can drop into their website. This project demonstrates practical AI deployment for community impact.
